Kode Huffman

Salah satu algoritma yang biasa digunakan dalam kompresi data adalah algoritma Pengkodean Huffman.Algoritma ini dikembangkan oleh David A. Huffman pada tahun 1952 yang merupakan salah satu mahasiswa MIT. Pada Algoritma pengkodean Huffman simbol yang mempunyai probabilitas paling besar diberi kode paling pendek (jumlah bit kode sedikit) dan simbol dengan probabilitas paling kecil akan memperoleh kode paling panjang (jumlah bit kode banyak). Kode tersebut diperoleh dengan cara memyusun sebuah pohon Huffman untuk masing-masing simbol berdasarkan nilai probabilitasnya.

Algoritma pohon Huffman dimulai dengan membuat dua buah node dengan frekuensi paling kecil berdasarkan daftar simbol dan probabilitas. Kemudian buatlah node parent dari node tersebut dengan bobot parent merupakan jumlah dari probabilitas kedua node anak tersebut. Selanjutnya masukkan node parent tersebut beserta bobotnya ke dalam daftar, dan kemudian kedua node anak beserta probabilitasnya dihapus dari daftar. Salah satu node anak dijadikan jalur (dilihat dari node parent) untuk pengkodean 0 sedangkan lainnya digunakan untuk jalur pengkodean 1.
